fontSize¶

字体大小插件

Configs¶

Editor.Plugin.fontSize.config.items¶
Editor.Plugin.fontSize.config.width¶

Events¶

无

Commands¶

  • fontSize()
  • queryFontSizeActive()

Config Details¶

Editor.Plugin.fontSize.config.items

{Object[]} - 可选。字体大小下拉菜单的名称和值集合，默认包括 “8px”,”10px” 等。例如：

[
    //ie 不认识中文？？？
    {
        name:"较大",
        value:"20px"
    },
    {
        name:"较小",
        value:"10px"
    }
]
Editor.Plugin.fontSize.config.width

{String} - 可选。下拉菜单的宽度，默认 “55px”.

Commands Details¶

Editor.commands.fontSize()¶
fontSize(fontSize, [apply=true])
对选区文字设置指定的字体大小
Parameters:
 
  • fontSize (String) – 指定字体大小值.
  • apply (Boolean) – 应用或取消指定的大小.

实例：

editor.execCommand("fontSize","10px"); //=> 设置选区文字大小为10px.
Editor.commands.queryFontSizeActive()¶
queryFontSizeActive(value, element)
查看指定元素是否被设置了对应大小
Parameters:
 
  • value (String) – 指定的大小值，例如 “10px”
  • element (KISSY.Node) – 编辑器中的某个元素
