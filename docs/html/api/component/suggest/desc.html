KISSY.Suggest - 功能描述¶

  1. 提示补全基本功能
  2. 完全跨域
  3. 小巧精简，仅依赖 ks-core, 压缩后不超过 9k
  4. 支持所有 A 级浏览器
  5. cache 功能
  6. 支持键盘控制：上下选择及回车后直接提交，ESC 键关闭
  7. 支持鼠标控制：鼠标选择和点击提交功能
  8. [DELAY] 支持匹配文字加亮
  9. [DELAY] 动画效果
  10. [DELAY] 在提示层中显示第一个搜索结果
  11. [DELAY] 整合本地表单的提示记录
  12. [DELAY] 关键词的模糊匹配提示功能
