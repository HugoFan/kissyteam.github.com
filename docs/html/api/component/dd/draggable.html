Class Detail¶

class KISSY.DD.Draggable(config)¶
Arguments:
 
  • config (Object) – 实例化可拖放对象的配置项, 详细见下节.

Config Attributes Detail¶

node¶

{String|HTMLElement} - 将要进行拖放的节点.

handlers¶

{Array<String|HTMLElement>} - 作为鼠标在其上按下时触发节点拖放的钩子. 如果不设置, 则整个 node 作为触发钩子.

Note

handlers 的每个元素 DOM 节点必须位于配置项 node DOM 子树中.

cursor¶

{String} - 默认值 “move”, handlers 元素中的每个元素要设置的鼠标样式.

mode¶

{String} - 枚举值, 默认值 “point”, 和 Droppable 关联, 决定何时和可放对象开始交互（触发相应事件）, 可取值 “point”,”intersect”,”strict”

  • 在 “point” 模式下, 只要鼠标略过可放对象, 即开始和可放对象交互.
  • 在 “intersect” 模式下, 只要拖动对象和可放对象有交集, 即开始和可放对象交互.
  • 在 “strict” 模式下, 只有拖动对象完全位于可放对象内, 才开始和可放对象交互.

Properties Detail¶

POINT¶

static, {String} - 等于 “point”

INTERSECT¶

static, {String} - 等于 “intersect”

STRICT¶

static, {String} - 等于 “strict”

node

{KISSY.Node} - 表示当前拖动的节点, 在应用 DD.Proxy 时表示代理节点.

dragNode¶

{KISSY.Node} - 表示配置项中 Draggable() 中 config.node 的值.

Note

实例属性通过 get 方法获取, 例如 drag.get("node")

Events Detail¶

dragstart(ev)¶

当可拖放对象开始被用户拖放时触发, 传给事件处理函数参数为事件对象 event .

Arguments:
 
  • ev (Object) – 自定义事件传递的参数
  • ev.event.drag (Object) – 自身, 当前拖放对象
drag(ev)¶

当可拖放对象拖放过程中触发, 传给事件处理函数为事件对象 event .

Arguments:
 
  • ev (Object) – 自定义事件传递的参数
  • ev.event.left (Number) – 拖放节点应该设置的相对文档根节点的横坐标位置.
  • ev.event.top (Number) – 拖放节点应该设置的相对文档根节点的纵坐标位置.
  • ev.event.pageX (Number) – 当前鼠标的绝对横坐标
  • ev.event.pageY (Number) – 当前鼠标的绝对纵坐标
  • ev.event.drag (Object) – 自身, 当前拖放对象
dragend(ev)¶

当用户鼠标弹起放弃拖放时触发 , 传给事件处理函数参数为事件对象 event.

Arguments:
 
  • ev.event.drag (Object) – 自身, 当前拖放对象
dragenter(ev)¶

同 dropenter , 只不过该事件在当前 Draggable 对象上触发.

dragover(ev)¶

同 dropover , 只不过该事件在当前 Draggable 对象上触发.

dragexit(ev)¶

同 dropexit , 只不过该事件在当前 Draggable 对象上触发.

dragdrophit(ev)¶

同 drophit , 只不过该事件在当前 Draggable 对象上触发.

dragdropmiss(ev)¶

当用户鼠标弹起但是没有放置当前 Draggable 对象到一个 Droppable 对象时触发. 传给事件处理函数参数为一个事件对象 event

Arguments:
 
  • ev.event.drag (Object) – 自身, 当前拖放对象

Note

Draggable 实例化后仅表示会根据鼠标拖放触发相应的事件, 但具体怎么处理仍需要调用者自己控制, 例如可监听 drag 事件, 根据事件对象参数的坐标设置拖放节点的具体位置.

new Draggable({node :"#d"}).on("drag",function(ev){
    this.get("node").offset({left:ev.left,top:ev.top});
});
