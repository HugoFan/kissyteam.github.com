DOM.hasProp()¶
Boolean hasProp ( selector , propertyName )
判断符合选择器的第一个元素是否含有特定 property 属性.
Parameters:
 
  • selector (string|HTMLCollection|Array<HTMLElement>) – 字符串格式参见 KISSY selector
  • propertyName (string) – property 名称
Returns:
 

符合选择器的第一个元素是否含有特定 property 属性.
Return type:
 

Boolean

Note

当你使用该方法时, 请先考虑下是否是自定义 property , 如果是自定义 property 推荐采用 DOM.data() 系列方法. 如果是 DOM property , 一般直接判断 DOM.prop() 返回值即可
