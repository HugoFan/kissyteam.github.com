.. currentmodule:: event

pinch
====================================

触屏上用双指调整某个 dom 元素大小时出现

定义
-----------------------

触屏上用双指调整某个 dom 元素大小时出现

包括格外下面的属性：

.. pinch.distance

    {Number} 双指的绝对距离

.. pinch.scale

    {Number} 双指相对于开始调整时具体的倍数
