Event.Object¶

class Event.Object(event)¶

对 dom 原生事件进行兼容性处理, API 基本遵循 DOM-Level-3-Events 规范. 该类用于 event 模块内部调用，实例传入绑定的事件处理器作为第一个参数.

Parameters:
 event (HTMLEvent) – dom 原生事件.

事件属性¶

下面的列表都是该类实例的属性，虽然有些随着不同的事件类别值会是 undefined :

事件属性
