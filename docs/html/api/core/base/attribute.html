Methods Detail¶

attribute.addAttr()¶
void addAttr ( name, attrConfig )
给宿主对象增加一个属性.
Parameters:
  • name (String) – 属性名
  • attrConfig (Object) – 属性配置信息, 支持下面的配置项:
  • attrConfig.value (String|Number) – 属性默认值
  • attrConfig.valueFn (Function) – 提供属性默认值的函数
  • attrConfig.setter (Function) – 写属性时的处理函数
  • attrConfig.getter (Function) – 读属性时的处理函数

Note

如果配置项中没有设置 value, 会调用 valueFn 函数获取默认值并赋给 value.

attribute.addAttrs()¶
void addAttrs ( attrConfigs, values )
批量添加属性.
Parameters:
  • attrConfigs (Object) – 属性名/配置信息对.
  • values (Object) – 属性名/值对, 该值为属性的默认值, 会覆盖配置信息中的默认值.
attribute.hasAttr()¶
void hasAttr ( name )
判断是否有名为 name 的属性.
Parameters:name (String) – 属性名
attribute.removeAttr()¶
void removeAttr ( name )
删除名为 name 的属性.
Parameters:name (String) – 属性名
attribute.set()¶
void set ( name, value )
设置属性 name 的值为 value.
Parameters:
  • name (String) – 属性名
  • value (String) – 属性的值
attribute.get()¶
void get ( name )
获取属性 name 的值.
Parameters:name (String) – 属性名

Note

当没有设置属性值时, 会取该属性的默认值.

attribute.reset()¶
void reset ( name )
重置属性 name 为初始值. 如果不给出属性名, 默认将所有属性名全部重置为初始值.
Parameters:name (String) – 属性名

Note

重置属性值, 同样会触发 beforeAttrNameChange() 和 afterAttrNameChange() 事件.

Events Detail¶

attribute.beforeAttrNameChange()¶
beforeAttrNameChange ()
名为 “attrName” 的属性, 在改变它的值之前触发该事件.
attribute.afterAttrNameChange()¶
afterAttrNameChange ()
名为 “attrName” 的属性, 在改变它的值之后触发该事件.

Demo¶

KISSY.ready(function(S) {
    // 自定义类
    function myClass() {
    }

    // 让你的类支持属性描述符
    S.augment(myClass, S.Attribute);

    var cls = new myClass();

    // 增加属性
    cls.addAttr('size', {
        value: 0,
        setter: function(v) {
            if (S.isString(v) && v.indexOf('inch')!== -1) {
                return parseFloat(v)*10/3;
            }
            return parseFloat(v);
        },
        getter: function(v) {
            return v;
        }
    });
    // 绑定事件
    cls.on('afterSizeChange', function(ev){
        console.log('change '+ ev.attrName + ': '+ev.prevVal+' --> '+ev.newVal);
    });

    // 设置属性
    cls.set('size', 20);

    // 获取属性
    alert(cls.get('size'));

    // 重置
    cls.reset();
    alert(cls.get('size'));
});
