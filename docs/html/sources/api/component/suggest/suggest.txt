Suggest

.. class:: KISSY.Suggest(textInput, dataSource, config)

    :param String|HTMLElement textInput: 输入框
    
    :param String dataSource: 数据源
    
    :param Object config: 配置项
    
        .. attribute:: containerCls 
        
            | 悬浮提示层的class 
            | 默认为空

            | 默认为空，和input输入框的宽度保持一致
