class component.Controller¶
Controller (config)
  • 继承自 Base , 包含其全部配置,属性,方法,事件.
  • 扩展自 ComponentProcess, 包含其全部配置，属性，方法，事件.
Parameters:config (Object) – 配置项，详细见下方 Configs Detail .

Configs Detail¶

Note

content 和 srcNode 不能同时都设置

Controller.config.content¶

{String} - 可选, 设置内容 html.

Controller.config.disabled¶

{Boolean} - 可选，该组件是否初始禁用.

Controller.config.elAttrs¶

{Object} - 可选，附加给组件根节点的属性键值对

Controller.config.elBefore¶

{KISSY.Node} - 可选，组件根节点的渲染到该节点之前.

Controller.config.elCls¶

{String} - 可选，附加给组件根节点的样式类

Controller.config.elStyle¶

{Object} - 可选，附加给组件根节点的内联样式

Controller.config.focused¶

{Boolean} - 可选，该组件是否初始获得焦点.

Controller.config.height¶

{Number} - 可选，组件的高度，单位像素

Controller.config.parent¶

{Controller} - 可选，该组件的父组件.

Controller.config.prefixCls¶
{String} - 可选，默认 “ks-” . 组件的 css 样式类前缀 . 例如假设组件为 menu ，则该组件内的样式类名为 {prefixCls}menu，默认为 “ks-menu”.
可用于实现自定义皮肤.
Controller.config.prefixXClass¶

{String} - 可选，组件 prefix 的超类。只在config中实用。当超类未被指定时，用这个做超类。

Controller.config.render¶

{KISSY.Node} - 组件要应用的节点。默认 S.all(“body”)，组件根节点的渲染为该节点最后一个节点.

Controller.config.srcNode¶

{KISSY.Node} - 可选，组件从页面中已存在的该节点中渲染而来.

Note

srcNode 时设置其他属性不起作用，属性通通在 html 标签中指定，并且 html 标签必须包含完整结构，例如 content 节点必须存在

Controller.config.visible¶

{Boolean} - 默认 true ，是否显示

Note

只是为组件的根节点添加/删除 {prefix}{component}-hidden 形式的 css class，自行指定具体的 css 样式。

Controller.config.width¶

{Number} - 可选，组件的宽度，单位像素

Controller.config.x¶

{Number} - 横轴位置

Controller.config.y¶

{Nunmber} - 纵轴位置

Controller.config.zIndex¶

{Number} - z-index 值
