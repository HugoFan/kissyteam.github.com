Events Detail¶

Droppable.dropenter()¶
dropenter ( ev )
当一个 Draggable 对象根据其 Draggable.mode 配置达到和当前 Droppable 实例交互条件时触发.
一般即鼠标进入当前 Droppable 对象代表节点的区域, 可简单理解成 mouseenter.
Parameters:
 
  • ev.drag (Object) – 当前交互的 Draggable 对象.
  • ev.drop (Object) – 自身, 当前Droppable对象.
Droppable.dropover()¶
dropover ( ev )
当一个 Draggable 在当前 Droppable 实例上移动时触发, 可简单理解成 mouseover.
Parameters:
 
  • ev.drag (Object) – 当前交互的 Draggable 对象.
  • ev.drop (Object) – 自身, 当前Droppable对象.
Droppable.dropexit()¶
dropexit ( ev )
当一个 Draggable 离开当前 Droppable 实例时触发, 可简单理解成 mouseleave.
Parameters:
 
  • ev.drag (Object) – 当前交互的 Draggable 对象.
  • ev.drop (Object) – 自身, 当前Droppable对象.

drophit ( ev )
