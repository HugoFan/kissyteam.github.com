append(node, parent)

将 node 追加到 parent 子节点最后.

Arguments:
 
  • node (string|HTMLElement) – 插入的节点. 字符串格式参见 KISSY selector , 获取匹配的第一个元素.
  • parent (string|HTMLElement) – 参照父节点. 字符串格式参见 KISSY selector , 获取匹配的第一个元素.
