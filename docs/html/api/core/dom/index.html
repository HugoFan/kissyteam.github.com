DOM¶

by 玉伯 , 承玉

在 1.2 版更改: DOM 不再接受 Node 以及 NodeList 类型的参数，DOM 只处理选择器以及原生 DOM 节点。 KISSY Node 以及 NodeList 对象上的相关处理可直接调用其自身方法。
