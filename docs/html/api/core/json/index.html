DEMO¶

注意 json 字符串的格式，属性必须要双引号括起来

KISSY.JSON.parse('{"x":1}'); // => ok
KISSY.JSON.parse("{'x':1}"); // => exception : SyntaxError
KISSY.JSON.parse("{x:1}"); // => exception : SyntaxError


KISSY.JSON.stringify({"x":1}); // => '{"x":1}'
KISSY.JSON.stringify({x:1}); // => '{"x":1}'
KISSY.JSON.stringify({'x':1}); // => '{"x":1}'
