KISSY.unEscapeHTML¶

1.2 新版功能.

KISSY.unEscapeHTML(str)¶

将字符串中的 html 实体字符替换成对应字符

参数:
 str (string) – 包含 html 实体字符的字符串
返回:
 替换实体字符后的字符串
返回类型:
 string

例如

KISSY.unEscapeHTML("&lt;a&gt;x&lt;/a&gt;"); // =>  "<a>x</a>"
