.. currentmodule:: event

pinchStart
====================================

触屏上开始用双指调整某个 dom 元素大小时出现

定义
-----------------------

触屏上开始用双指调整某个 dom 元素大小时出现

包括格外下面的属性：

.. attribute:: pinchStart.distance

    {Number} 开始时双指的绝对距离


.. attribute:: pinchStart.scale

    {Number} 固定为 1
