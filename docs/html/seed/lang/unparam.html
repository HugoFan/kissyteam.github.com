KISSY.unparam¶

KISSY.unparam(str)¶

将参数字符串 str 还原为对象。

Parameters:o (object) – 参数字符串
Returns:参数的对象表示
Return type:object

Changed in version 1.2: 数组处理改变，key 不需要加 [] 如 v=1&v=2 => {v:[1,2]}

例如

var S = KISSY;

S.unparam('foo=1&bar=2'); // => { foo: 1, bar: 2 }
S.unparam('foo=1&bar=2&bar=3'); // => { foo: 1, bar: [2, 3] }
