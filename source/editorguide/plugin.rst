.. _editorusage-plugin:


插件名称列表
--------------------

elementpaths
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    主要用户开发, 显示光标处所在dom路径.

    .. image:: ../_static/editorguide/elementpaths.gif


sourcearea
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    点击后可以查看编辑器产生的html代码.

    .. image:: ../_static/editorguide/sourcearea.gif

preview
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    点击后可以弹出新窗口, 查看编辑器的内容预览.

    .. image:: ../_static/editorguide/preview.gif


templates
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    点击后可以弹出模板选择窗口, 选定后可以将模板代码插入到编辑器光标所在处.

    .. image:: ../_static/editorguide/templates.gif

separator
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    间隔线.

    .. image:: ../_static/editorguide/separator.gif


undo
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    可以撤销重做.

    .. image:: ../_static/editorguide/undo.gif

removeformat
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    可以清除选择区域的编辑格式(字体, 大小, 加粗).

    .. image:: ../_static/editorguide/removeformat.gif

font
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    大小: 可以改变选择区域字体的大小.

    .. image:: ../_static/editorguide/fontsize.gif

    字体: 可以改变选择区域的字体种类.

    .. image:: ../_static/editorguide/fontfamily.gif

    粗体: 可以将选择区域文字加粗.

    .. image:: ../_static/editorguide/fontbold.gif

    斜体: 可以将选择区域文字倾斜.

    .. image:: ../_static/editorguide/fontitalic.gif

    下划线: 可以给选择区域文字加下划线.

    .. image:: ../_static/editorguide/fontunderline.gif

    删除线: 可以给选择区域文字加删除线.

    .. image:: ../_static/editorguide/fontdelete.gif

format
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    可以将光标所在处块加入标题格式.

    .. image:: ../_static/editorguide/format.gif

forecolor
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    设置选择区域的文本颜色.

    .. image:: ../_static/editorguide/forecolor.gif

bgcolor
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    设置选择区域的背景颜色.

    .. image:: ../_static/editorguide/bgcolor.gif

list
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    为选择区域或光标所在处加上项目编号.

    .. image:: ../_static/editorguide/list1.gif

    为选择区域或光标所在处加上列表编号.

    .. image:: ../_static/editorguide/list2.gif

indent
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    减少光标处的缩进量.

    .. image:: ../_static/editorguide/indent1.gif

    增加光标处的缩进量.

    .. image:: ../_static/editorguide/indent2.gif

justify
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    左对齐: 光标所在块左对齐.

    .. image:: ../_static/editorguide/justify1.gif

    居中对齐: 光标所在块居中对齐.

    .. image:: ../_static/editorguide/justify2.gif


    右对齐: 光标所在块右对齐.

    .. image:: ../_static/editorguide/justify3.gif

link
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    编辑选择区域的超链接.

    .. image:: ../_static/editorguide/link.gif

image
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    输入图像地址将图像插入到光标所在处.

    .. image:: ../_static/editorguide/image.gif

flash
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    输入flash地址将flash插入到光标所在处.

    .. image:: ../_static/editorguide/flash.gif


music
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    输入音乐地址将音乐插入到光标所在处.

    .. image:: ../_static/editorguide/music.gif


smiley
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    选择表情并将对应表情插入到光标所在处.

    .. image:: ../_static/editorguide/smiley.gif

table
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    输入表格相关参数并将对应表格插入到光标所在处.

    .. image:: ../_static/editorguide/table.gif

resize
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    可拖动调整编辑区域大小.

    .. image:: ../_static/editorguide/resize.gif

pagebreak
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    插入分页标记.

    .. image:: ../_static/editorguide/pagebreak.gif

maximize
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    将编辑器充满整个屏幕.

    .. image:: ../_static/editorguide/maximize.gif
