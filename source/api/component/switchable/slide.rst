子类 Slide
===================================================================


默认配置
-------------------------------------------------------------

S.Slide 接口及配置项, 与 :attr:`~Switchable.Switchable` 相同, 其中以下配置项的默认值有所区别:

**Switchable.autoplay**    是否自动切换, 默认为 true

**Switchable.circular**    是否循环切换, 默认为 true
