.. currentmodule:: event

rotate
====================================

触屏上用双指旋转某个 dom 元素时出现

定义
-----------------------

触屏上用双指旋转某个 dom 元素时出现

包括格外下面的属性：

.. attribute:: rotate.angle

    {Number} 当前双指的角度值

.. attribute:: rotate.rotation

    {Number} 双指和开始相比改变的角度值

------------------
