下拉菜单位置

概述和示例

可以通过 menu 配置控制下拉菜单的位置

实例

通过 menu 配置控制下拉菜单的位置

new SplitButton({
    render:"#para2",
    button:{
        xclass:'button',
        content:'action'
    },
    alignWithEl:false,
    menuButton:{
        xclass:'menu-button',
        matchElWidth:false,
        menu:{
            children:[
                {
                    content:'some action'
                },
                {
                    content:'more action'
                }
            ]
        }
    }
}).render();
