event.stopPropagation¶

返回 Event.Object

Object.stopPropagation()¶

停止事件沿 dom 树向上冒泡，组织祖先节点的所有事件处理器执行.我们可以使用 isPropagationStopped 来判断当前方法是否执行过. 该方法在 fire() 中也起作用。

Note

调用该方法不会组织该节点其他事件处理器的运行，如果要需要调用 stopImmediatePropagation()

例子¶

阻止 click 事件的冒泡

var $=KISSY.Node.all;
$("p").on('click',function(event){
  event.stopPropagation();
  // do something
});
