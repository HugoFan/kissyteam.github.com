KISSY.available¶

KISSY.available(id, fn)¶
Parameters:
  • id (string) – 页面元素 id
  • fn (function) – 回调函数，在 id 元素可用时立刻执行。

Note

当需要比 KISSY.ready() 反应更快的探测到某个元素可用时使用。
