flash
====================================

|  flash
|  这是 KISSY 的 Flash 模块, 包含以下子模块：
|     - flash-ua Flash 播放器版本信息, 并作为 KISSY.UA 的补充.
|     - flash-embed 静态注册/动态嵌入 swf 对象等方法, 是 Flash 的具体行为主体
